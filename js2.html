    34 What is variable typing in js?
    Ans:

    1. Dynamic Typing: JavaScript does not require you to declare the data type of a variable when you create it. Instead, the data type is determined dynamically based on the value assigned to the variable.

    2. Type Coercion: JavaScript performs type coercion, which means it will automatically convert values from one type to another in certain situations

    3. Type Inference: JavaScript infers the data type of a variable based on the value it currently holds. When you assign a value to a variable, JavaScript determines its data type accordingly.

    4. Explicit Type Conversion: While JavaScript handles many type conversions automatically, you can also explicitly convert values from one type to another using functions like parseInt(), parseFloat(), String(), Number(), etc.

    38 What is iief in js?
    Ans:

    "IIFE": ["Immediately Invoked Function Expression."]

    An IIFE is a JavaScript function that is defined and executed immediately after its creation.

    It is often used to create a private scope for variables and functions to prevent them from polluting the global scope.

    1. (function () { ... }): This part defines an anonymous function. The function is wrapped in parentheses to make it an expression rather than a declaration. This is necessary because in JavaScript, a function declaration must have a name, but in this case, we don't need one.

    2. (...) immediately invokes (calls) the function. This means that as soon as the JavaScript engine encounters the closing parenthesis, it executes the function.

    40 What is a pure function?
    Ans:

    A Pure Function is a function (a block of code) that always returns the same result if the same arguments are passed.

    1. Deterministic: Given the same input, a pure function will always return the same output.

    2. No Side Effects: A pure function does not have any side effects, such as modifying global variables, mutating data structures

    The impureAdd function is not pure because it modifies the result variable outside of its scope, which is a side effect.
